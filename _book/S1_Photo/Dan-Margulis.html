PPW Workflow

PPW's "Decompositio Philosophy" is process "Luminosity" and "Color" separately mostly through the time.

  1. Color
    1. Color Correction：
      • RGB Curve Adjustment Layer in RGB mode ---> CH3
    2. Color Range Define
      • Define "Black Point" and "White Point" in RGB mode ---> CH3\&CH4
  2. Luminosity
    1. Luminosity Reallocation
      • Luminosity layer by "Channel Mix" (Apply Image or Calculation) ---> CH4
      • Luminosity Curve Adjustment Layer
    2. Contrast Boost
      • Luminosity Curve Adjustment Layer ---> CH4
    3. Extract Detail of Shadow \& Highlight
      • PS Shadows/Hightlights
      • PPW S/H+OK ---> CH7
      • PPW Bigger Hammer ---> CH10
    4. Strengthen Specific Things
      • PPW Darken Sky SC and Darken Sky B ---> CH8
    5. Extend Luminocity Range
      • Screen and Multiply ---> CH12
  3. Color Again
    1. ??
      • PPW H-K ---> CH13
    2. Skin Color Retouch
      • PPW Skin Desaturation ---> CH9
    3. Color Boost
      • PPW CB ---> CH5
      • PPW MMM ---> CH6
    4. Define Color "Black Point" and "White Point"
      • ?? ---> ALL-CH?
    5. Color ease
      • PS "Mark" ---> ALL-CH?
      • PS "Opacity" ---> ALL-CH?
  4. Sharpen
    1. Sharpen
      • PS USM Sharpen
      • PS Smart Sharpen
      • PPW sharpen 2015
      • PPW Sharpen old
  5. Ending
    1. Versions Blending
      • Layer blending mode ---> CH2
      • Layer opacity
      • Layer blend if
      • Layer mask
    2. Color Space match
      • Color transfer
